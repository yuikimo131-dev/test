# 其味无穷 - Endless Flavor

A web application that generates endless flavor combinations for culinary exploration. Discover unique pairings that will inspire your cooking and expand your palate.

## Features

- **Endless Combinations**: Generate unique flavor pairings from a curated database of ingredients
- **Beautiful Interface**: Modern, responsive design with smooth animations
- **Save Favorites**: Keep track of your favorite flavor combinations
- **Statistics**: Track how many combinations you've generated and saved
- **Local Storage**: Your saved combinations persist between sessions
- **Keyboard Shortcuts**: Press Space/Enter to generate, 'S' to save

## How It Works

The app combines primary flavors (like chocolate, vanilla, basil) with secondary flavors (like sea salt, bacon, truffle oil) to create unique culinary pairings. Each combination comes with a description explaining why the flavors work well together.

## Technology Stack

- **Frontend**: Pure HTML5, CSS3, and JavaScript (no frameworks required)
- **Storage**: LocalStorage for data persistence
- **Design**: Responsive design with CSS Grid and Flexbox
- **Animations**: CSS transitions and keyframe animations

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Click "Generate New Combination" to discover flavor pairings
4. Save your favorites for later reference

## Flavor Database

The app includes a curated database of:
- **15 Primary Flavors**: Base ingredients like chocolate, vanilla, garlic, basil
- **15 Secondary Flavors**: Complementary ingredients like sea salt, bacon, truffle oil

Each flavor includes descriptive information to help users understand its characteristics and pairing potential.

## Keyboard Shortcuts

- **Space/Enter**: Generate new combination
- **S**: Save current combination (when not typing in input fields)

## Browser Compatibility

This application works in all modern browsers that support:
- ES6 JavaScript features
- CSS Grid and Flexbox
- LocalStorage API
